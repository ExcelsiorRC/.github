# Excelsior Running Club

A competitive, race-focused running club in San Francisco. Structured track workouts, serious training, and a real team since 1973.

**Group Runs**

| Day | Run | Time & Location |
|-----|-----|-----------------|
| Tue & Fri | Dawn Patrol | 5:45 AM · Conservatory of Flowers |
| Wed | Track Workout | 6:00 PM · Kezar Stadium |
| Weekend | Long Run | Varies · Organized on Discord |

No signup required for your first run. Just show up.
